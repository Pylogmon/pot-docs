# Google

由于众所周知的原因，国内连接不上 Google 服务器，所以 pot 提供了两种解决方案。

## 方法 2：使用镜像站地址

pot 提供了自定义域名设置，可以在设置中设置自定义域名，通过设置自定义域名为你所搭建的镜像站地址，可以实现不使用代理的情况下连接 Google 服务器。

## 方法 1：开启代理

在常规设置中设置软件内代理。

将 Google 服务设置中的自定义域名留空，使用官方地址。
